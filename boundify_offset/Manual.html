Boundify offset plug-in v0.8.15

This plugin can
  • Create outlines/offset line of shapes(Rectangle,Line,Circle,Elipse,Path)
  • Line caps and corner caps sytles
  • Minimal line profile
  • Path Effects
  • Rounded dash-array
  • Dot to dot
  • Stamp topmost group-shape

How to use

Outline command
  1. Select a shape(s) in VectorLayer
  2. Use this plug-in(from Tools Menu >Outline)
  3. The shape convert to Outlined shape
Boundify Offset... command
  1. Select a shape(s) in VectorLayer
  2. Use this plug-in(from Tools Menu >Dialog open and setting
  3. Choose line styles (or Move Path)
  4. Preview , OK button
  5. Append Offseted shape(s)

Support Line Join Styles

  • miter
  • bevel
  • round

Support Line Cap Styles(Start/End)

  • butt
  • square
  • round
  • arrow (Add an arrowhead to either line cap)
  • ribbon(Add a ribbon-style to either line cap)
  • brush
  • brush2 (+ Distorted brush effect)
  • brush3
  • bell
  • sigma

Optional feature

  • FillView : Offset Path with fill color
  • Use Original Path : Simple effect to a path (Not create outline)
  • Stroke : Offset Path with red-stroke-color, Uncheck for fill

Line Effects

  • none : Default (Offseted line)
  • linear : Tapered Line
  • linear_a : Tapered Line(Alternative)
  • organic : Floral streamline
  • chain : Line Crossing
  • gear : by square wave
  • sawtooth :Saw-tooth-ed line / Pointed Zigzag
  • punk : Create a burst balloon or Waved line
  • fluffy : Create a cloud-like shape or Bubble-shaped line
  • rough : Create Rough-edged line
  • dot_to_dot : Plot numberd dots(very slowly)
  • round_dot_dash :Rounded doted line
  • dynamic_dash : Create dash line dynamical
  • random_dash : For cyber/flat design
  • stamp_top_groups : Stamp topmost(z-index) group-shapes along the path
